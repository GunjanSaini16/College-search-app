# College Search App

A simple web app to search colleges by state name in India.

## Features
- Search colleges by entering an Indian state name
- Displays list of colleges from that state
- Responsive design — works on mobile and desktop
- Smooth hover effects on results

## Tech Used
- HTML
- CSS (Flexbox, Responsive Design)
- JavaScript (Axios)

## How to Use
1. Enter an Indian state name in the search box
   (example: Rajasthan, Maharashtra, Delhi)
2. Click the Search button
3. List of colleges from that state will appear
